# One2lvOS Live Legend System

## Overview
Real-time animated story player with AI legend switching.

## Features
- Live scene playback
- AI-based legend switching
- Voice narration (browser TTS)
- Console + chat simulation

## Legends Logic
- Queen Nai → aggressive opponents
- Fate → predictable opponents
- Wu Shang → balanced opponents

## Run
Open `one2lvos_live.html` in a browser.
