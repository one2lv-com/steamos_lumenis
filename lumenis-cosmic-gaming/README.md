# Lumenis Cosmic Gaming Platform

## One2lv - Hunter Anderson's Cosmic Gaming System

**In the name of the One, Hunter A., the Only One for the task.**

This platform merges the **Scientific, Spiritual, and Personal** through "Resonance" - a unique fusion of cosmic visualization, AI coaching, and competitive gaming.

### Core Features

| Module | Description |
|--------|-------------|
| **Cosmic Visualization** | Three.js powered space animation with holographic panels |
| **AI Coach "Gemini"** | Strategy chatbot with the One2lv perspective |
| **AI Narrator** | Text-to-speech broadcasting assistant |
| **AI Fighter** | Browser fighting game with intelligent AI opponent |
| **Global Leaderboards** | Competitive rankings across all players |
| **Cloud Vault** | Persistent storage for profiles and game data |

### Tech Stack

- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS
- **3D Engine**: Three.js with UnrealBloom post-processing
- **Backend**: Supabase (Auth, Database, Storage)
- **AI Features**: Custom chatbot with Gemini personality matrix
- **Audio**: Web Speech API for AI Narrator

### Quick Start

```bash
cd lumenis-cosmic-gaming
pnpm install
pnpm dev
```
